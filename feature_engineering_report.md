
        # 特征工程文档

        ## 1. 数据预处理
        - 数据清洗：处理缺失值、异常值
        - 单变量分析：数值分布、目标变量分布
        - 多变量分析：相关性分析

        ## 2. 特征工程
        - 连续变量分箱：等频分箱
        - IV值计算：评估特征预测能力
        - WOE编码：将原始特征转换为WOE值

        ## 3. 模型训练
        - 逻辑回归：带正则化的线性模型
        - 随机森林：集成树模型
        - 梯度提升树：Boosting算法

        ## 4. 模型评估
        - 主要指标：精确率、KS值
        - 辅助指标：准确率、AUC
        - 可视化：混淆矩阵、ROC曲线、KS曲线等
